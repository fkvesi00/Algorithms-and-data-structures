Napiši program koji generira N slučajnih brojeva i sprema ih u niz. Generirani niz treba sortirati 
primjenom svih slijedećih algoritama: 
a) Exchange sort 
b) Selection sort 
c) Insertion sort 
d) Bubble sort 
Za svaki algoritam potrebno je ispisati i vrijeme potrebno se sortira niz.

Write a program that generates N random numbers and stores them in a string. The generated string needs to be sorted
using all of the following algorithms:
a) Exchange sort
b) Selection sort
c) Insertion sort
e) Bubble sort
For each algorithm it is necessary to print and the time required to sort the string.
